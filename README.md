Fake News Detector is a Machine Learning-based project that classifies news articles as Real or Fake using NLP techniques like TF-IDF and Logistic Regression. The model is trained on labeled datasets and achieves high accuracy after proper preprocessing and feature extraction.

This project uses Natural Language Processing (NLP) and Machine Learning to detect whether a news article is real or fake.

### 🔧 Tech Stack:
- Python
- Scikit-learn
- NLTK
- TF-IDF Vectorizer
- Logistic Regression

### 🎯 Goal:
To classify news articles accurately and reduce misinformation using ML techniques.
